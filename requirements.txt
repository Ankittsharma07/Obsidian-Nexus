Obsidian Nexus - Runtime Requirements
-------------------------------------
1. Node.js 20.x (LTS) and npm 10.x or higher installed locally.
2. PostgreSQL database (Supabase recommended) with:
   - Connection pooling URL (port 6543) exported as DATABASE_URL.
   - Direct connection URL (port 5432) exported as DIRECT_URL for Prisma migrations.
3. NextAuth secrets:
   - NEXTAUTH_SECRET generated via `openssl rand -hex 32`.
   - NEXTAUTH_URL pointing to your local or deployed domain.
4. Optional integrations:
   - SUPABASE_URL / SUPABASE_KEY for REST access.
   - GEMINI_API_KEY for AI insights (omit for fallback mode).
   - REDIS_URL if using external caching/rate limiting.
5. Local setup steps:
   - `npm install`
   - `npx prisma generate`
   - `npx prisma migrate dev`
   - `npx prisma db seed`
   - `npm run dev`

Keep all actual secret values out of version control.***
